# **Dominios problemática**

**Nombre:** Fabián Matus Alarcón

**Dominios:**

* **Principal:** Ingeniería de datos  
* **Complementario:** Visualización de datos  
* **Remedial:** Inteligencia Artificial (IA)

El dominio principal del proyecto es Ingeniería de Datos, ya que constituye el núcleo técnico para resolver la imprecisión y el riesgo del cálculo volumétrico mediante el procesamiento de información espacial obtenida a través de drones.

Se selecciona Visualización de Datos porque aporta una interfaz clara para la lectura de los resultados del modelado 3D y las métricas en terreno, mejorando significativamente la integración operativa para la toma de decisiones.

Finalmente, se incorpora Inteligencia Artificial con el propósito de fortalecer nuestra brecha en el conocimiento técnico de modelos analíticos, evidenciando su aplicación mediante la exploración de algoritmos de visión computacional para automatizar y mejorar la detección y segmentación en las capturas fotográficas aéreas.

Esta combinación resulta coherente para entregar un sistema automatizado de cálculo volumétrico integral para el control del inventario forestal.
